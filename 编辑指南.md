# SLDX_Ambition_Website 编辑指南

## 📖 前言

本指南面向没有编程基础的团队成员，教你如何轻松更新战队网站内容。

---

## 🛠️ 准备工作（只需配置一次）

### 1. 安装必要软件

- **VS Code** (推荐)
  - 下载地址：https://code.visualstudio.com/
  - 安装后打开，搜索并安装插件：`Markdown Preview Enhanced`

- **Git**
  - 下载地址：https://git-scm.com/zh-cn
  - 安装时保持默认选项即可

### 2. 配置 Git 账号

打开终端（Mac）或 Git Bash（Windows），输入以下命令：

```bash
git config --global user.name "你的名字"
git config --global user.email "你的 GitHub 邮箱"
```

### 3. Fork 仓库

1. 打开 https://github.com/SYLU-Ambition/SLDX_Ambition_Website
2. 点击右上角 **Fork** 按钮
3. 复制你自己的仓库地址（`https://github.com/你的用户名/SLDX_Ambition_Website.git`）

### 4. 克隆仓库到本地

```bash
# 选择一个文件夹存放代码
cd ~/Documents  # Mac/Linux
# 或
cd D:\Projects  # Windows

# 克隆你的 fork 仓库
git clone https://github.com/你的用户名/SLDX_Ambition_Website.git

# 进入项目目录
cd SLDX_Ambition_Website
```

## ✏️ 编辑页面内容

### 方式一：在线编辑（推荐新手）

适合快速修改文字内容，无需安装任何软件。

1. 打开 GitHub 仓库
2. 进入 `doc` 文件夹
3. 找到要编辑的页面（如 `about.md`）
4. 点击右上角的 **铅笔图标** ✏️
5. 直接编辑内容
6. 在页面底部填写修改说明，点击 **Commit changes**

### 方式二：使用 VS Code 编辑（推荐）

适合大量修改或批量更新。

#### 步骤 1：打开项目

```bash
cd SLDX_Ambition_Website
code .  # 用 VS Code 打开项目
```

#### 步骤 2：找到要编辑的文件

所有页面都在 `doc/` 目录下：

```
doc/
├── index.md          # 首页
├── about/            # 关于我们
│   ├── intro.md      # 队伍介绍
│   └── members.md    # 成员介绍
├── news/             # 新闻动态
├── competition/      # 比赛相关
└── ...
```

#### 步骤 3：编辑 Markdown 文件

- 点击任意 `.md` 文件即可编辑
- 按 `Ctrl+Shift+V` (Mac: `Cmd+Shift+V`) 可预览效果
- 语法说明见下方「Markdown 基础语法」

#### 步骤 4：提交并推送

在 VS Code 左侧点击 Git 图标（或按 `Ctrl+Shift+G`）：

1. 点击文件名旁边的 **+** 号（暂存更改）
2. 在输入框填写修改说明（如 "更新成员介绍"）
3. 点击 **提交** 按钮（✓）
4. 点击 **同步更改** 或 **推送** 按钮（↑）

或者使用终端：

```bash
# 添加所有修改
git add .

# 提交更改
git commit -m "修改说明：更新了 XX 内容"

# 推送到远程仓库
git push
```

## 📝 添加新页面

### 步骤 1：创建新文件

在 `doc/` 目录下新建 `.md` 文件，如 `doc/news/2025-04-18-competition.md`

### 步骤 2：添加页面头部信息

```markdown
---
title: 2025 年电赛获奖新闻
description: 描述信息
---

# 2025 年电赛获奖新闻

这里是正文内容...
```

### 步骤 3：在侧边栏添加链接

编辑 `.vitepress/sidebar.mjs`，在对应位置添加：

```javascript
{
  text: '2025 年电赛获奖新闻',
  link: '/news/2025-04-18-competition',
}
```

## 🔧 常用操作

### 拉取最新内容

在编辑之前，先拉取队友的最新修改：

```bash
git pull
```

### 查看修改历史

```bash
git log --oneline
```

### 撤销修改

```bash
# 撤销未提交的修改
git checkout -- 文件名

# 撤销已提交的最后一次提交
git reset --hard HEAD~1
```

## ❓ 常见问题

### Q: 修改后网站没有更新？

A: 检查是否完成了 `git push`，推送后 GitHub Actions 会自动构建部署，大约需要 1-2 分钟。

### Q: 如何预览修改效果？

A: 
- VS Code 中按 `Ctrl+Shift+V` 预览
- 或在本地运行 `pnpm run dev` 实时预览（需要安装 Node.js 和 pnpm）

### Q: 编辑冲突了怎么办？

A: 先 `git pull` 拉取最新内容，解决冲突后再提交。

### Q: 如何回退到之前的版本？

A: 在 GitHub 仓库的 **Insights** → **Deployments** 中可以查看历史部署记录，或联系管理员。
